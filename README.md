# geo-clj

clojure geocode library

## Usage

### geocode an address:
(geo-clj.google/geocode "123 main street Fairhaven MA 02719")

### calculate distance between two distances
(geo-clj.distance/simple {:lat 41.0 :lng -70} {:lat 41.1 :lng -70.2})
